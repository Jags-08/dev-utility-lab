# Security Policy

## Supported Versions

Only the most recent major versions are supported with security updates. 

| Version | Supported          |
| ------- | ------------------ |
| >= 0.1.x| :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please do NOT report it in the public issue tracker.

Instead, please send an email to the repository maintainer directly or use GitHub's private vulnerability reporting feature.
We will respond within 48 hours to coordinate a structured patch and CVE assignment.
