# Roadmap

This document outlines the future development plan for `dev-utility-lab`. 

## 🎯 v0.2.0 (Planned)
- Include directory-level operations (e.g. recursive file finders).
- Improve logging functionalities natively within the utilities.
- Add text-formatting CLI tools (e.g. Markdown to HTML).

## 🎯 v0.3.0 (Planned)
- Implement configuration file support (`~/.dev-utils.json`) to persist CLI preferences.
- Add advanced statistics capabilities to `math_ops`.

## 🎯 v1.0.0 (Long-Term)
- Stabilize API for production integrations.
- Auto-generate extensive API documentation via Sphinx/MkDocs.
- Support Windows/macOS native executable builds via PyInstaller.
