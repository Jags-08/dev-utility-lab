# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-05-10
### Added
- Initial release of the `dev-utility-lab` toolkit.
- Command-line interface (`dev-utils`) built with `argparse`.
- Math module (`add`, `factorial`, `gcd`, `lcm`, `fibonacci`, `is-prime`).
- Strings module (`reverse-string`, `palindrome-check`).
- Random module (`generate-password`, `random-int`, `shuffle-list`).
- Data module (`read-json`, `read-file`).
- Full CI/CD pipeline using GitHub Actions.
- Developer setup instructions and contributing guidelines.
