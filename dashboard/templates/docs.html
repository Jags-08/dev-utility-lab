Reference documentation for integrating the dev-utility-lab securely directly through API calls without opening the browser sandbox UI.

Endpoint: POST /api/run-tool

Securely executes functions mapped inside the internal Python utility library and returns structured computational responses.

Request Format

{
    "tool_id": "math/gcd",
    "payload": {
        "a": 48,
        "b": 18
    }
}

Response Format

{
    "success": true,
    "result": 6,
    "execution_time_ms": 0.0084
}

Error Response

{
    "success": false,
    "error": "Invalid type for a. Expected int."
}
